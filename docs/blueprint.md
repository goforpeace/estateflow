# **App Name**: EstateFlow

## Core Features:

- Project Creation & Management: Create and manage real estate projects with details like name, location, developer/landowner shares, start date, and status.
- Flat Inventory: Maintain an inventory of flats for each project, specifying flat number, ownership (developer/landowner), sale price, and status (available, sold, reserved).
- Customer & Installment Tracking: Assign customers to flats, record booking money and monthly installments, and track total received, total due, and payment history. Payment types: cash and bank.
- Cash Inflow Logging: Log all cash inflows, categorizing by project, flat, customer, payment type (booking/installment), date, and amount.
- Cash Outflow Logging: Log all cash outflows by project or office, categorized by expense type (material, labor, utility, office), supplier/vendor, amount, and date.
- Office Cost Allocation: Allocate monthly office expenses to projects using either equal allocation or project size ratio.
- AI-Powered Cash Flow Forecasting Tool: The AI tool predicts potential cash shortages or surpluses for each project based on historical data and payment schedules. It helps make informed decisions to mitigate risks.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to evoke trust, stability, and professionalism, essential for financial applications.
- Background color: Light gray (#F9FAFB), creating a clean, neutral backdrop that enhances readability and reduces eye strain.
- Accent color: Bright orange (#EA580C) to draw attention to key actions and important data points, like CTAs or alerts, injecting a sense of urgency and focus.
- Body and headline font: 'Inter' sans-serif for a modern, machined, objective, neutral look; suitable for headlines or body text.
- Clean, professional admin dashboard layout with clear data presentation and easy navigation.
- Use simple, consistent icons for project status, payment types, and expense categories.
- Subtle transitions and animations to enhance user experience when navigating between modules or updating data.
